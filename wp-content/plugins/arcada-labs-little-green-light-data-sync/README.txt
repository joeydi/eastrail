## Description

CRM Sync for Little Green Light is a powerful plugin built for nonprofits. It enables you to synchronize your donor information, transactions and WordPress user data with Little Green Light to keep your CRM up-to-date. This plugin comes with a free version which allows for WordPress user syncing, while the premium version includes synchronization with WooCommerce and/or Gravity Forms.

## Features

- Synchronize your transaction, donor and WordPress user data with Little Green Light.
- Keep all the changes in the CRM system up-to-date in real-time.
- Automate your workflow and ensure the efficiency of your nonprofit organization.
- Simple and easy to use interface with detailed documentation.

## Installation

To install and activate the plugin, follow these simple steps:

1. Download the plugin from the WordPress repository or from our [website](https://clacks.arcadalabs.com/arcada-labs-little-green-light-data-sync.zip).
2. Upload the plugin ZIP file to the `/wp-content/plugins/` directory on your WordPress site.
3. Activate the plugin from the 'Plugins' menu in WordPress.

## Frequently Asked Questions

### What is included in the free version of the plugin?

The free version of the plugin allows you to synchronize your WordPress user data with Little Green Light CRM.

### What payment options do you offer for the premium version?

We offer multiple payment options including Paypal, Credit Card and Bank Transfer.

### Which versions of WooCommerce and Gravity Forms are compatible with the plugin?

The plugin is compatible with the latest versions of WooCommerce and Gravity Forms.
